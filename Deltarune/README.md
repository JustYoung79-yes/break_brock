# DELTARUNE - 어두운 세계

델타룬 스타일의 브라우저 RPG입니다.

## 실행 방법

1. 이 폴더에서 로컬 서버를 띄우거나 `index.html`을 브라우저에서 직접 엽니다.
2. **아무 키나 눌러** 타이틀을 지나 오버월드로 들어갑니다.
3. **방향키**로 캐릭터(크리스)를 움직입니다.
4. **초록 사각형(NPC)**에 닿으면 라세이 대화가 나옵니다. **Z / 스페이스 / 엔터**로 다음 문장.
5. **보라색 원(적)**에 닿으면 전투가 시작됩니다.
   - 전투 중 **방향키**로 빨간 영혼(소울)을 움직여 보라색 탄을 피하세요.
   - **FIGHT**: 적을 쓰러뜨리고 전투 종료.
   - **ACT**: 행동 (대사만).
   - **SPARE**: 자비를 베풀고 전투 종료.
   - **ITEM**: 아이템 (현재 없음).
6. HP가 0이 되면 자동으로 1로 복구되고, 잠시 후 필드로 돌아갑니다.

## 파일 구성

- `index.html` - 게임 페이지
- `style.css` - 픽셀/다크 테마 스타일
- `game.js` - 오버월드, 전투, 대화 로직

## 로컬 서버로 실행 (권장)

```bash
npx serve .
# 또는
python -m http.server 8080
```

이후 브라우저에서 `http://localhost:3000` (또는 8080)으로 접속하세요.
